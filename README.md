#  NITK Photography Club Website

This project is a **Photography Club Website for NITK Surathkal** built using **HTML, CSS, and JavaScript**.

The website allows visitors to explore the club, view the photo gallery, and learn about activities.
Club members can log in to manage events and upload photographs.

# Features

 1. Home Page

The homepage introduces the NITK Photography Club.

Features:

* Hero section with background image
* Club title and tagline
* Member login option
* Navigation bar
* Footer with club information


### 2. About Page

The About page explains the purpose and activities of the club.

Includes:

* Vision of the photography club
* Club activities
* Community involvement
* Photography events and workshops


3. Gallery Page

Displays photographs captured by club members.

Features:

* Responsive photo grid layout
* Automatically loads photos from storage
* Default gallery images included
* Visitors can freely view all images

 4. Member Dashboard

Accessible only to club members.

Members can:

* Add event information
* Upload photographs
* Delete photographs
* Manage content displayed in the gallery

** Photo Management**

Photos uploaded by members are stored using **LocalStorage**.

* Display uploaded photos
* Delete photos
* Photos automatically appear in the public gallery



 Technologies Used

* HTML5
* CSS3
* JavaScript
* LocalStorage (for storing events and photos)
